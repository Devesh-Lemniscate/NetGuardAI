══════════════════════════════════════════════════════════════
  🎯 HOW IT WORKS
══════════════════════════════════════════════════════════════

BACKEND (realtime_analyzer.py):
  1. Captures packets continuously
  2. Extracts 27 features every 5 seconds
  3. Runs ML prediction
  4. Writes to JSON files (thread-safe)
  
DATA EXCHANGE (file-based):
  • ml_predictions.json (last 1000 predictions)
  • ml_stats.json (live statistics)
  • Updated after each window

FRONTEND (app_realtime.py):
  1. Reads JSON files every 1-2 seconds
  2. Updates all metrics and charts
  3. Shows threat alerts
  4. Auto-refreshes display
